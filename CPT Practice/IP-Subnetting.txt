## 🌐 IP ADDRESSING & SUBNETTING – ALL IN ONE (HINDI NOTES)

### ⭐⭐⭐ IP Address

An IP Address is a **logical address** used to uniquely identify a device on a network.
It works at **Network Layer (Layer 3 – OSI Model)** ⭐⭐
IPv4 uses **32 bits (4 octets)** ⭐⭐⭐
Example: `192.168.1.1`

---

### ⭐⭐⭐ Types of IP Address

* **IPv4** – 32-bit address ⭐⭐⭐
* **IPv6** – 128-bit address ⭐⭐
* **Public IP** – Used on Internet ⭐⭐⭐
* **Private IP** – Used in LAN ⭐⭐⭐
* **Static IP** – Manual assignment ⭐⭐
* **Dynamic IP** – Assigned by DHCP ⭐⭐

---

### ⭐⭐⭐ IANA & ICANN

* **IANA** allocates IP addresses globally ⭐⭐⭐
* **ICANN** manages domain names and IP policies ⭐⭐

---

### ⭐⭐⭐ Classful IP Addressing

IP addresses are divided into **five classes: A, B, C, D, E** ⭐⭐⭐

| Class | Range   | Default Mask        | Use                 |
| ----- | ------- | ------------------- | ------------------- |
| A     | 1–126   | 255.0.0.0 (/8)      | Large networks ⭐⭐⭐  |
| B     | 128–191 | 255.255.0.0 (/16)   | Medium networks ⭐⭐⭐ |
| C     | 192–223 | 255.255.255.0 (/24) | Small networks ⭐⭐⭐  |
| D     | 224–239 | Multicast ⭐⭐        |                     |
| E     | 240–255 | Research ⭐          |                     |

---

### ⭐⭐⭐ Class A, B, C Key Points

* **Class A Hosts**: 16,777,214 ⭐⭐
* **Class B Hosts**: 65,534 ⭐⭐
* **Class C Hosts**: 254 ⭐⭐⭐

---

### ⭐⭐ Class D & E

* **Class D** → Multicast communication ⭐⭐
* **Class E** → Experimental / Research ⭐

---

### ⭐⭐⭐ Public vs Private IP Address

Private IP ranges (Not routable on Internet) ⭐⭐⭐

* **Class A**: `10.0.0.0 – 10.255.255.255`
* **Class B**: `172.16.0.0 – 172.31.255.255`
* **Class C**: `192.168.0.0 – 192.168.255.255`

---

### ⭐⭐⭐ Reserved IP Addresses

* **APIPA**: `169.254.0.0/16` (No DHCP) ⭐⭐⭐
* **Loopback**: `127.0.0.1` (Testing) ⭐⭐⭐
* **Network Address**: First IP ⭐⭐
* **Broadcast Address**: Last IP ⭐⭐

---

### ⭐⭐⭐ Subnet Mask

Subnet Mask separates **Network ID and Host ID** ⭐⭐⭐
Example:
`255.255.255.0 = /24`

---

### ⭐⭐⭐ Subnetting

Subnetting means **dividing one network into multiple smaller networks** ⭐⭐⭐
Benefits:

* Efficient IP usage ⭐⭐⭐
* Improved security ⭐⭐
* Reduced broadcast traffic ⭐⭐

---

### ⭐⭐⭐⭐⭐ Subnetting Math (Decimal to Binary)

Binary weights:
`128 64 32 16 8 4 2 1` ⭐⭐⭐⭐⭐

---

### ⭐⭐⭐⭐ CIDR (Classless Inter-Domain Routing)

CIDR removes class boundaries ⭐⭐⭐⭐
Example:
`192.168.1.0/26`

---

### ⭐⭐⭐⭐⭐ Subnetting Formulas

* **Number of Subnets** = `2ⁿ` ⭐⭐⭐⭐⭐
* **Number of Hosts** = `2ʰ − 2` ⭐⭐⭐⭐⭐

---

### ⭐⭐⭐⭐⭐ Class C Subnetting

Most commonly used in exams & interviews ⭐⭐⭐⭐⭐
Default mask: `/24`

---

### ⭐⭐⭐⭐ Class B Subnetting

More host bits, same calculation rules ⭐⭐⭐⭐

---

### ⭐⭐⭐⭐⭐ VLSM (Variable Length Subnet Mask)

Allows **different subnet sizes in same network** ⭐⭐⭐⭐⭐
Used in real-world networks and routing ⭐⭐⭐⭐

---

### ⭐⭐⭐ Supernetting

Supernetting combines multiple networks into one ⭐⭐⭐
Opposite of subnetting
Used in routing tables ⭐⭐⭐

Here are **ALL important definitions in ONE LINE EACH**, simple and clear:

* **IP Address**: A logical address used to uniquely identify a device on a network.
* **IPv4**: A 32-bit IP address written in decimal format (e.g., 192.168.1.1).
* **Public IP**: An IP address that is accessible over the internet.
* **Private IP**: An IP address used inside a local network and not routable on the internet.
* **Static IP**: An IP address that is manually assigned and does not change.
* **Dynamic IP**: An IP address automatically assigned by DHCP.
* **IANA**: The organization that allocates IP addresses globally.
* **ICANN**: The organization that manages domain names and IP address policies.
* **Classful Addressing**: An IP addressing method that divides IPs into fixed classes A, B, and C.
* **Class A IP**: An IP class used for very large networks with default mask /8.
* **Class B IP**: An IP class used for medium networks with default mask /16.
* **Class C IP**: An IP class used for small networks with default mask /24.
* **Class D IP**: An IP class reserved for multicast communication.
* **Class E IP**: An IP class reserved for experimental and research use.
* **Subnet Mask**: A value used to separate the network part and host part of an IP address.
* **Subnetting**: The process of dividing one network into multiple smaller networks.
* **CIDR**: A classless IP addressing method that uses prefix notation like /24.
* **APIPA**: An automatic private IP assigned when DHCP is unavailable.
* **Loopback Address**: An IP address (127.0.0.1) used to test the local system.
* **Network Address**: The first IP in a subnet that identifies the network.
* **Broadcast Address**: The last IP in a subnet used to communicate with all hosts.
* **VLSM**: A subnetting technique that allows different subnet sizes in one network.
* **Supernetting**: The process of combining multiple networks into a larger network.
